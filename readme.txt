== Description ==

This Buddypress plugin enables users to send attachments in private messages. The type of file types allowed and maximum size of the attachement can be controlled by admin.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Then from the navigation panel go to Buddypress > Message Attachement to configure the file types and size
